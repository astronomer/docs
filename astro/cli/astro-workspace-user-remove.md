Remove a user from your current Workspace.

## Usage

```sh
astro workspace user remove
```

## Options

| Option    | Description                                | Possible Values |
| --------- | ------------------------------------------ | --------------- |
| `<email>` | The email for the user you want to remove. | Any valid email |
